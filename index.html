Problem A2

Description

We're given an RLC Circuit, and we have to find out (approximately) the current i{i}i in the main (and only) branch, and charge q{q}q on the capacitor.

Initially the capacitor has a charge q=1C{q = 1C}q=1C, which then discharges to set up a current in the branch. To calculate current we apply Kirchoff's Loop Law in the branch by assuming a direction of the current.

Applying the Loop Law we get the following set of differential equations to work with.

iR+Ldidt+qC=0{iR+L\frac{di}{dt}+\frac{q}{C}=0}iR+Ldtdi​+Cq​=0 dqdt=i{\frac{dq}{dt}=i}dtdq​=i
where,
 i  =current in the branch{i \;= \text{current in the branch}}i=current in the branch
 L=inductance{L = \text{inductance}}L=inductance
 R=resistance{R = \text{resistance}}R=resistance
 C=capacitance{C = \text{capacitance}}C=capacitance
 q  =charge on capacitor{q \;= \text{charge on capacitor}}q=charge on capacitor

Solving by Euler's Method

We solve the seet of differential equations iteratively. Replacing, dqdt=qn+1−qnΔt{\frac{dq}{dt}=\frac{q_{n+1} - q_{n}}{\Delta t}}dtdq​=Δtqn+1​−qn​​ and, didt=in+1−inΔt{\frac{di}{dt}=\frac{i_{n+1} - i_{n}}{\Delta t}}dtdi​=Δtin+1​−in​​ The equations reduce to,
qn+1=qn+inΔt{q_{n+1} = q_n + i_n\Delta t}qn+1​=qn​+in​Δt in+1=in(1−RlΔt)−qnΔtLC{i_{n+1} = i_n\left(1-\frac{R}{l}\Delta t\right) - \frac{q_n\Delta t}{LC}}in+1​=in​(1−lR​Δt)−LCqn​Δt​ where Δt{\Delta t}Δt is the step size of time. In the given problem,
Δt=0.01s{\Delta t = 0.01s}Δt=0.01s
t0=0s{t_0 = 0s}t0​=0s
tfinal=0.1s{t_{final} = 0.1s}tfinal​=0.1s

MATLAB Code

clear all;              % clear all variables
clc;                    % clear command window
close all;              % close open windows

% constants
R = 200                 % resistance in ohms
L = 5                   % inductance in Henry
C = 10^(-4)             % capacitance in Faraday

n = 11;                 % num of steps 
t = zeros(n,1);         % Time in seconds
I = zeros(n,1);         % Current in Ampere
q = zeros(n,1);         % Charge in Coloumb

% initial conditions
t(1) = 0;
I(1) = 0;
q(1) = 1;

dt = 0.01;              % size of time step in sec

% Iterative solution derived from Euler's Method
for i = 2:n
    t(i) = t(i-1) + dt;
    q(i) = q(i-1) + I(i-1)*dt;
    I(i) = I(i-1) + ( - (I(i-1)*R)/L - q(i-1)/(L*C) )*dt;
end

% Plots
figure;
plot(t, I, '-b','LineWidth', 3);
title('Current vs Time');
xlabel('Time (in sec)');
ylabel('Current (in Amperes)')

plot(t, q, '-r', 'LineWidth', 2);
title('Charge on Capacitor vs Time');
xlabel('Time (in sec)');
ylabel('Charge (in Coloumbs)')

Plot

Observation

Initially, the current in the opposite (w.r.t. the direction assumed) direction increases, and the charge on capacitor decreases. The electrical energy of capcitor C{C}C transforms into the magnetic energy of inductor L(∵VL=Ldidt){L \left(\because V_L = L\frac{di}{dt}\right)}L(∵VL​=Ldtdi​). So, with time the Inductor is charged, and Capacitor discharged i.e. q=0{q = 0}q=0. Now, inductor becomes the power source, the direction of current reverses, and capcitor begins to accumulate charge (but on opposite plates, and that's the reason for negative charge depicted in plot.) However, we notice that the amplitude of current, i{i}i and charge q{q}q have dropped (∣i∣{|i|}∣i∣ goes from 35.2 A to 16.3 A, while ∣q∣{|q|}∣q∣ goes from 1 C to 0.47 C). The mentioned drops are observed because a fraction of power is being dissipated in the Resistor (aka as copper losses). And, thus energy oscillates between L{L}L and C{C}C until the resistor dissipates all of the energy supplied in system at t=0{t=0}t=0.

Problem T1

Description

A coffe cup (initially at 68°C) is kept in a room at 21°C.
We use Euler's method to find the numerical solution (by substituting the derivatives with truncated taylor expansions.)

The differential equation in question is the Newton's Law of Cooling.
dTdt=−k(T−Ta)\frac{dT}{dt} = -k(T-T_a)dtdT​=−k(T−Ta​)
where,
  T=T =T= temperature of Cup
  Ta=T_a =Ta​= temperature of Surroundings
  k=k =k= Cooling Coefficient
  t=t =t= time

Euler's method

So substituting, dTdt=T(i+1)−T(i)Δt\frac{dT}{dt} = \frac{T(i+1) - T(i)}{\Delta t}dtdT​=ΔtT(i+1)−T(i)​ in the above equation, where Δt=1\Delta t =1Δt=1 is the step size of time (in minutes). The equation reduces to, T(i)=T(i−1)+k(Ta−T(i−1))ΔtT(i) = T(i-1) + k(T_a - T(i-1))\Delta tT(i)=T(i−1)+k(Ta​−T(i−1))Δt

Plot

To observe the efficiency of Euler's method, the Temperature has also been calculated analytically at different instants.

Observation

Euler's method works sufficiently well, even when the step size chosen in our case was big (10% of total time plotted).

MATLAB Code

*Note: Code also present in the root folder, named as tut1_1.m

clear all;  % clear all variables
clc;        % clear command window
close all;  % close open windows

% constants
k = 0.017;   % per min
T_a = 21;    % deg Celsius

n = 11;              % num of steps 
t = zeros(n,1);    
Texact = zeros(n,1);   % exact solution
T = zeros(n,1);        % numerical solution

% initial conditions
t(1) = 0;
T(1) = 68; 
Texact(1) = 68;

dt = 1;      % size of time step in min

for i = 2:n
    t(i) = t(i-1) + dt;
    Texact(i) = T_a + (T(1) - T_a)*exp(-k*t(i));
    T(i) = T(i-1) + k*dt*T_a - k*dt*T(i-1);
end

plot(t, Texact, 'og','LineWidth', 3);
title('Temperature of Coffee Cup as a Function of Time');
xlabel('Time (in min)');
ylabel('Temperature of cup (in Celsius)')
hold on;
plot(t, T, '-r', 'LineWidth', 2);
legend('Analytical',"Euler's Method");
